Technology

Infrastructure built for reliability

Cloudflare Workers
Bot logic runs on Cloudflare's global edge network — sub-millisecond execution, zero cold starts, 99.9%+ uptime. No single point of failure.
OKX API v5
Direct integration with OKX's latest REST and WebSocket API. Real-time orderbook data, instant order placement, and live position monitoring.
Cloudflare KV
User data, bot configurations, and trade history are stored in Cloudflare KV — a globally distributed key-value store with low-latency reads worldwide.
Resend
Transactional emails — verification codes, subscription confirmations, trade alerts — delivered reliably via Resend's developer-first email API.
