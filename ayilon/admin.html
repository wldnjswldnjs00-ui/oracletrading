키 설정 방법 (1회만)
  1. OKX > 어필리에이트 > API 에서 읽기 전용 키 생성 (Key / Secret / Passphrase)
  2. Cloudflare 대시보드 > Workers & Pages > ayilon-email > Settings > Variables and Secrets
  3. Secret 3개 추가: OKX_AFFILIATE_KEY, OKX_AFFILIATE_SECRET, OKX_AFFILIATE_PASS
  4. 저장 후 이 페이지 새로고침 → 위 스위치가 활성화됩니다.
